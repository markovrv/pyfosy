# FOS Importer - Финальная версия GUI

## Описание

Проект предоставляет графический интерфейс для автоматизации импорта вопросов в систему ФОС ВятГУ. Создано несколько версий для разных потребностей.

## Версии GUI

### 1. gui_simple.py - Простая версия (работает во всех окружениях)
- ✅ Не использует потоки
- ✅ Совместимость с ограничивающими средами
- ✅ Все основные функции интерфейса
- ✅ Симуляция процесса импорта
- 🔧 Кнопка "Авторизация" для получения списков
- 🔧 Поля ввода логина/пароля
- 🔧 Выбор из выпадающих списков
- 🔧 Тестовый режим

### 2. gui_complete.py - Полная версия (для продакшена)
- 🚀 Полная интеграция с pyppeteer
- 🚀 Реальная авторизация в системе ВятГУ
- 🚀 Автоматическая загрузка данных
- 🚀 Полный цикл импорта
- ⚠️ Требует стабильного окружения

## Запуск

### Простая версия (рекомендуется для демонстрации)
```bash
cd pyfosy
python gui_simple.py
```

### Полная версия (для реального использования)
```bash
cd pyfosy
python gui_complete.py
```

### Через launcher (с проверкой зависимостей)
```bash
cd pyfosy
python launch_gui.py
```

## Инструкции по использованию

### Шаг 1: Запуск приложения
Выберите подходящую версию и запустите её

### Шаг 2: Ввод данных
1. Введите **логин** и **пароль** для системы ВятГУ
2. Выберите **GIFT-файл** с вопросами

### Шаг 3: Авторизация
1. Нажмите кнопку **"Авторизация"**
2. Дождитесь загрузки списков кафедр, специальностей и предметов

### Шаг 4: Выбор параметров
1. Выберите **кафедру** из выпадающего списка
2. Выберите **специальность** из выпадающего списка
3. Выберите **предмет** из выпадающего списка

### Шаг 5: Импорт
1. Нажмите кнопку **"Начать импорт"**
2. Следите за прогрессом в журнале операций
3. Дождитесь завершения импорта

## Файлы проекта

- `gui_simple.py` - Простая версия GUI (рекомендуется)
- `gui_complete.py` - Полная версия с pyppeteer
